# Dokumentation v1

Willkommen zur **Version 1.0** Dokumentation.

> **Hinweis**: Diese Version ist veraltet. Wir empfehlen das Upgrade auf [v2](/de/docs/v2/) für die neuesten Funktionen und Verbesserungen.

## Überblick

v1 war die erste stabile Version von shipyard und bietet:

- Grundlegende Page Builder Funktionalität
- Blog-Komponenten
- Dokumentations-Layouts
- Tailwind CSS Integration

## Erste Schritte

1. [Installationsanleitung](/de/docs/v1/installation) - v1 einrichten
2. [Konfiguration](/de/docs/v1/configuration) - Projekt konfigurieren

## Upgrade

Um auf die neueste Version zu upgraden:

1. Lies die [v2 Migrationsanleitung](/de/docs/v2/migration)
2. Aktualisiere deine Abhängigkeiten
3. Folge den Migrationsschritten

## Support

v1 befindet sich im Wartungsmodus:

- Nur kritische Sicherheitsfixes
- Keine neuen Features geplant
- Community-Support verfügbar

Für das beste Erlebnis, upgrade bitte auf [v2](/de/docs/v2/).
