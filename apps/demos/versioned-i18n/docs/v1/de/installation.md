# Installationsanleitung (v1)

> **Hinweis**: Dies ist die Dokumentation für v1. Für die neueste Version, siehe die [v2 Installationsanleitung](/de/docs/v2/installation).

Diese Anleitung behandelt die Installation von v1.

## Voraussetzungen

- Node.js 16.x oder höher
- npm 8.x oder höher

## Schnellstart

```bash
# Neues Projekt erstellen
npm create astro@latest mein-projekt

# shipyard Pakete installieren
npm install @levino/shipyard-base@1 @levino/shipyard-docs@1

# Entwicklungsserver starten
npm run dev
```

## Konfiguration

Siehe die [v1 Konfigurationsanleitung](/de/docs/v1/configuration) für Setup-Optionen.
