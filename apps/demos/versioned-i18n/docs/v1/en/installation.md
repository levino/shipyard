# Installation Guide (v1)

> **Note**: This is documentation for v1. For the latest version, see the [v2 installation guide](/en/docs/v2/installation).

This guide covers installing v1.

## Prerequisites

- Node.js 16.x or higher
- npm 8.x or higher

## Quick Start

```bash
# Create a new project
npm create astro@latest my-project

# Install shipyard packages
npm install @levino/shipyard-base@1 @levino/shipyard-docs@1

# Start development server
npm run dev
```

## Configuration

See the [v1 Configuration Guide](/en/docs/v1/configuration) for setup options.

## Upgrading

Consider upgrading to v2 for:

- Better performance
- TypeScript support
- New components

See the [Migration Guide](/en/docs/v2/migration) for details.
