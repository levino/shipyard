# About Single Language Demo

This is a demonstration site for **single-language shipyard configurations**.

## What is This?

This demo showcases how to build a website with shipyard that doesn't need internationalization. It's perfect for:

- Personal blogs
- Company websites targeting one market
- Documentation sites for single-language products
- MVP and prototype sites

## Key Features

- **No i18n overhead**: Simpler configuration and faster builds
- **Clean URLs**: Direct paths without language prefixes
- **Easy migration**: Can add i18n later when needed
- **Full feature set**: All shipyard features without complexity
