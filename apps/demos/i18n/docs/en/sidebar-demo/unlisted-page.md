# Unlisted Page

This page is marked as `unlisted: true` in the frontmatter.

## What does unlisted mean?

- The page **is rendered** and accessible via direct URL
- The page **does not appear** in the sidebar navigation
- The page **does not appear** in pagination (prev/next links)

## Use cases

Unlisted pages are useful for:
- Internal documentation that shouldn't clutter the navigation
- Draft content that needs to be shared via direct link
- Landing pages for specific campaigns or features
- Documentation for specific user groups
