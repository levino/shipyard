# Benutzerdefiniertes Label

Dieses Element hat eine benutzerdefinierte Seitenleistenbeschriftung im Frontmatter definiert.

Obwohl es alphabetisch an erster Stelle steht (`custom-label.md`), erscheint diese Seite **zweiter** in der Seitenleiste, weil sie einen höheren `sidebar_position`-Wert (30) hat als Benutzerdefinierte Klasse (10).

```yaml
sidebar_label: Fancy Label ✨
sidebar_position: 30
```
