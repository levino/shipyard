# Contributing

Thank you for your interest in contributing to shipyard! We welcome contributions from the community.

## Getting Started

Before contributing, please familiarize yourself with our coding standards:

- [Code Style Guide](/en/docs/contributing/code-style) - Our comprehensive guide covering functional programming principles, naming conventions, testing requirements, and more.

## Communication

All communication in issues, pull requests, and discussions should be in English.
