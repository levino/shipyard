# Mitwirkung

Wir erwarten von allen Mitwirkenden, dass sie auf Englisch kommunizieren. Dies erleichtert die Zusammenarbeit in unserem internationalen Team und stellt sicher, dass alle Diskussionen für alle Beteiligten zugänglich sind.

Aus diesem Grund ist dieser Bereich der Dokumentation nicht ins Deutsche übersetzt. Bitte lesen Sie die englische Version unter [Contributing](/en/docs/contributing).
